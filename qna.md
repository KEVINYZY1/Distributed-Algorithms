1.为什么会出现多种全局排序？请举例说明。


```text
由于不同 process 的 event 可能会有相同的 timestamp，例如 E1 = `<3:7>`, E2 = `<3:5>`。
如果 P7 < P5 的话， E1 => E2。
如果 P5 < P7 的话， E2 => E1。
```
